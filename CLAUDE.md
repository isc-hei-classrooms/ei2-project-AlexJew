This folder contains files for building and running a net electricity load (consumption - production) forecasting model. This folder is structured into multiple sub-folders including:

- others: Stores all of the other documents related to the management of the project
- data: Stores the data files
- claude: stores information, context and memory specific to claude
- notebooks: stores the marimo notebooks
- utils: stores additional utility python scripts

Instructions

- Write all relevant information to retain context inside markdown files in the folder "claude"
- Keep track of the project to-dos via the table stored in the section https://www.notion.so/alexandre-jewell/Energy-load-and-production-forecast-for-OIKEN-312c3942dace804982cbef88a35fbbac?source=copy_link#31ec3942dace801ab6dff0e3e180d331
- After each change of the python file, propose a git commit message and ask whether you should commit the change
- Do not make large changes, instead make small incremental changes focused on a specific objective and commit it right afterwards
